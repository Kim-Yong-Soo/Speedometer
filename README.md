# 간편 속도계
- 간단 설명: 속도 측정 어플


* [Google Play Store](https://play.google.com/store/apps/details?id=com.kys.speedometer)

- 주요 기능
  - 속도 측정 (현재 속도, 최고 속도, 평균 속도)
  - 경로 기록
- 목표
  - 열차 이용 시 다음 역까지의 거리 표시 (현재 위치 기반)
  - 이동 경로 저장 및 이전 경로와의 비교 결과 제공
  - 속도, 고도 그래프로 제공
  - 다크 모드 완전 적용
  - 다양한 언어 지원
  - 초기화 버튼
